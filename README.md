# motion_detector
RPi motion detection with OpenCV

# Installation
run `install_opencv.sh`

# Config
```json
{
  "show_video": true,
  "video_resolution": [360, 240],
  "min_upload_seconds": 3.0,
  "min_motion_frames": 8,
  "camera_warmup_time": 2.5,
  "delta_thresh": 5,
  "resolution": [640, 480],
  "fps": 16,
  "min_area": 5000
}
```
